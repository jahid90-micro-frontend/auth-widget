# Auth App

A react based app to manage user auth.

## Development

### Clone the repository

```sh
$ git clone git@github.com:jahid90-micro-frontend/auth-widget.git
```

### Build

```sh
$ just build
```

### Start a dev server

```sh
$ just docker:start
```
